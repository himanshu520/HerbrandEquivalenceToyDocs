◆ Instructions

std::vector<InstructionTy> Program::Instructions

Instructions in the program.

The first and last element of this vector are dummy which stands for START and END instructions. They facilitates branching and confluence at the start and the end of the program respectively.

◆ Variables

MapVector<std::string> Program::Variables

Variables used in the program.

It is a MapVector object which maps variable names to unique integers, so that we finally have to work only with integers instead of working with strings. Also MapVector maps unique strings to unique integers, which also helps to identify same variable used at different places.
